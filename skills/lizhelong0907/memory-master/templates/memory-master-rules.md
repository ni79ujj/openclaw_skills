# 记忆系统规则 - memory-master v2.5.3

## 写入规则
- 写入时机：讨论有结论立刻写记忆 → 更新索引
- 学会新技能时立刻写知识库 → 更新索引
- 实施与否都记录

## 文件位置
- 记忆索引：memory/daily-index.md
- 记忆文件：memory/daily/YYYY-MM-DD.md
- 知识库索引：memory/knowledge-index.md
- 知识库文件：memory/knowledge/*.md

## 格式
### 记忆格式
```
## [日期] 主题
- 因：原因/背景
- 改：做了什么
- 待：待办
```

### 索引格式
- 记忆索引：`- 主题 → daily/日期.md,日期.md`
- 知识库索引：关键字列表

## 搜索规则
1. 用户有要求 → 执行
2. 检查上下文
3. 搜索索引
4. 找到对应项 → 读取执行
5. 没有 → 网络学习 → 写入知识库 → 更新索引
